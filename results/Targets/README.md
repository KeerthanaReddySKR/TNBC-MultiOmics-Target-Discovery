# Final Therapeutic Target Outputs

This directory contains the final prioritized therapeutic targets identified by the TNBC multi-omics pipeline.

Key files:

- **Final_high_confidence_targets.tsv**  
  Top predicted therapeutic targets.

- **Final_Therapeutic_Target_Panel.tsv**  
  Final candidate gene panel used for downstream analysis.

- **All_candidates_with_evidence.tsv**  
  Complete candidate list including supporting evidence from differential expression, network analysis, mutation integration, and machine learning.

- **Drug_Target_Map.tsv**  
  Mapping between candidate genes and potential drug targets.

- **Experimental_Validation_Plan.tsv**  
  Proposed experimental strategies for validating predicted targets.
